# Vive Aux Philippines

Vive Aux Philippines aims to provide real, reliable information about the Philippines.

## Coding Expectations
- Performance optimized
- Structured and organized, following good coding practices
- SEO optimized
- Tested

## Tests
- Run `npm test` to execute the lightweight HTML checks (`tests/check.js`) that verify key SEO/accessibility hooks and script loading.

## AI Usage Policy
Any AI assistant must never commit, push, pull, delete, or perform similar actions unless explicitly instructed by the project owner.
